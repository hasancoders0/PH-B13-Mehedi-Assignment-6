#  Premium Digital Tools

## Project Description

Premium Digital Tools is a modern React-based web application where users can explore and purchase digital products like AI tools, templates, and productivity software. The app provides a smooth user experience with cart functionality, dynamic UI, and real-time notifications.

---

##  Technologies Used

*  React.js
*  Tailwind CSS + DaisyUI
*  React Toastify
*  Vite
*  JavaScript (ES6+)

---

## Features

*  Add to Cart System with real-time updates
*  Dynamic Product Rendering from JSON data
*  Toast Notifications for user actions (add/remove/checkout)
